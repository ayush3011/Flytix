# Flytix - (Flight Management System)

A robust web application for managing flights, airports, bookings, passengers, and schedules. Built with React, TypeScript, and Redux Toolkit.

## Features

- User authentication (login/signup) with local storage persistence
- Advanced state management using Redux Toolkit
- Utilizes React Router v6 for dynamic routing and route protection
- Modular architecture with reusable components
- Responsive UI with Bootstrap 5
- CRUD operations for flights, airports, bookings, passengers, and schedules
- Data persistence using local storage
- Error boundaries and loading states for better UX
- Code splitting and lazy loading for performance optimization
- Type safety throughout using TypeScript

## Getting Started

### Prerequisites

- Node.js (v14+)
- npm

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/ayush3011/Flytix.git
   cd ./Flytix
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. Open [http://localhost:{port}](http://localhost:port) in your browser.
   Or simply click the URL in the terminal.

## Project Structure
